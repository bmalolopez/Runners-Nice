Configuración inicial con Supabase · una sola vez 🔧

1Ve a supabase.com, inicia sesión y crea un nuevo proyecto.
2En tu proyecto Supabase, ve a SQL Editor y ejecuta este script (cópialo completo):
-- RunnersBR · Ejecuta esto en SQL Editor de Supabase CREATE TABLE IF NOT EXISTS config ( id TEXT PRIMARY KEY, admin_password TEXT DEFAULT 'runners2024' ); INSERT INTO config (id, admin_password) VALUES ('main', 'runners2024') ON CONFLICT (id) DO NOTHING; CREATE TABLE IF NOT EXISTS runners ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), number TEXT UNIQUE NOT NULL, first_name TEXT DEFAULT '', last_name TEXT DEFAULT '', nickname TEXT NOT NULL, birth TEXT DEFAULT '', color TEXT DEFAULT '', food TEXT DEFAULT '', phrase TEXT DEFAULT '', e_name TEXT DEFAULT '', e_phone TEXT DEFAULT '', joined TEXT DEFAULT '', gmm TEXT DEFAULT 'no', hospital TEXT DEFAULT '', created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS activities ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), rid UUID, type TEXT DEFAULT '', detail TEXT DEFAULT '', pts INTEGER DEFAULT 0, date TEXT DEFAULT '', note TEXT DEFAULT '', created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS chat ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), rid UUID, name TEXT DEFAULT '', msg TEXT DEFAULT '', date TEXT DEFAULT '', created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS photos ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), rid UUID, rname TEXT DEFAULT '', img TEXT DEFAULT '', cap TEXT DEFAULT '', act TEXT DEFAULT '', date TEXT DEFAULT '', created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS promos ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), rid UUID, rname TEXT DEFAULT '', img TEXT DEFAULT '', biz TEXT DEFAULT '', description TEXT DEFAULT '', dis TEXT DEFAULT '', con TEXT DEFAULT '', date TEXT DEFAULT '', approved BOOLEAN DEFAULT FALSE, created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE config DISABLE ROW LEVEL SECURITY; ALTER TABLE runners DISABLE ROW LEVEL SECURITY; ALTER TABLE activities DISABLE ROW LEVEL SECURITY; ALTER TABLE chat DISABLE ROW LEVEL SECURITY; ALTER TABLE photos DISABLE ROW LEVEL SECURITY; ALTER TABLE promos DISABLE ROW LEVEL SECURITY; ALTER PUBLICATION supabase_realtime ADD TABLE chat; CREATE TABLE IF NOT EXISTS avisos ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), title TEXT DEFAULT '', body TEXT DEFAULT '', date TEXT DEFAULT '', created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE avisos DISABLE ROW LEVEL SECURITY; -- Si ya tenías la base de datos creada, ejecuta esto también: ALTER TABLE runners ADD COLUMN IF NOT EXISTS gmm TEXT DEFAULT 'no'; ALTER TABLE runners ADD COLUMN IF NOT EXISTS hospital TEXT DEFAULT ''; ALTER TABLE promos ADD COLUMN IF NOT EXISTS approved BOOLEAN DEFAULT FALSE;
3Ve a Project Settings → API y copia la Project URL y la clave anon public.

Tus datos viven en tu propio proyecto Supabase. Gratis hasta 500MB.

🚀 Publicar en Vercel (opcional)
Sube este archivo RunnersBR.html a un repositorio GitHub (como index.html) y conéctalo en vercel.com → New Project. Cada corredora entra a la URL de Vercel y hace la configuración de Supabase una sola vez en su dispositivo.
